# Mind App -> Crie Sua Nuvem de Tags usando Ruby o Rails

Esse repositório é baseado no ruso MindApp - Desenvolvendo uma Cloud Tags com Ruby na Digital Inovation One, para o Bootcamp da Impulso Network.

Assuntos que o curso cumpriu:

* Ambiente de Desenvolvimento

* Teoria MVC

* Como criar seus controlers e views na mão

* Como usar o generator do Rails pra criar os models

* Como cirar relacionamentos importantíssimos entre models

* Como adicionar gems importantes para o projeto para agilizar o desenvolvimento

Muitos outros assuntos foram cobertos nesse que é, na minha opinião, o melhor curso da DIO, seguido pelo curso de "Conceitos e melhores práticas com o bando de dados POstgreSQL" e "Linux: A introdução ao sitema operacional"
